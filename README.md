# kash.fyi

---

My personal website! Built with Astro and Tailwind CSS.

Currently under development! You'll be able to view this site at https://www.kash.fyi/.

## Tech Stack

- React JS
- Astro 5
- Tailwind CSS
- GSAP (Handles animations)

## Development

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:4321)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
